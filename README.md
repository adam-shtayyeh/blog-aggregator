# Gator - CLI Blog Aggregator

Gator is a command-line interface (CLI) application built with **TypeScript** and **Node.js** to aggregate and manage RSS feeds periodically. It allows users to register, follow their favorite feeds, and automatically scrape articles, storing them efficiently in a **PostgreSQL** database using **Drizzle ORM**.

---

## Prerequisites

Make sure you have the following tools installed on your machine before running the project:
* **Node.js** (v18 or higher)
* **PostgreSQL** (running locally or hosted in the cloud)
* **npm** (Node Package Manager)
